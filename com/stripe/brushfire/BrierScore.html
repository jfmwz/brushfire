BrierScore

case class BrierScore[A, B](predicted: Map[(A, B), Double], actual: Map[(A, B), Long], counts: Map[(A, B), Long], totalCount: Long) extends Product with Serializable

BrierScore is a container for tracking the Brier score decomposition of a set of errors.

See http://www.eumetcal.org/intralibrary/open_virtual_file_path/i2055n15861t/english/msg/ver_prob_forec/uos2/uos2_ko2.htm for more information.

A

the classes to be predicted into.

B

the probability distribution bins.

predicted

every time an example is falls into a (predicted class, predicted probability bin), that probability is added here.

actual

counts the number of examples per (predicted class, predicted probability bin) that actually fall into the predicted class.

counts

the total number of examples in each (predicted class, predicted probability bin).

totalCount

the total number of examples.

  19. def reliability: Double

    Reliability measures how close forecast probabilities are to the true probabilities, and is defined as

    Reliability measures how close forecast probabilities are to the true probabilities, and is defined as

    reliability = 1 / (total # of examples) * sum_{class a} sum_{bin b} [ (# of examples that are predicted to fall into class a with probability in bin b) * (predicted probability - % of these examples that actually fall into class a)^2 ]

    1 / (total # of examples) * sum_{class a} sum_{bin b} [ (weighted count of examples that fall into class a with probability in bin b, where weights are the probability - # of these examples that actually fall into bin b)^2 / (# of examples that are predicted to fall into class a with probability in bin b)]

  20. def resolution: Double

    Resolution measures how much the conditional probabilities for each <class, bin> differ from the overall class probabilities, and is defined as
